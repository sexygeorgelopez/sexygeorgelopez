MonteMC

MonteMCs are V-Neck Piantas. They are a type of MonteM.

MonteMGs, the Broom Piantas, wear the same V-Necks as MonteMCs, so all of these values will work for them.

The V-Necks can be changed to be various colors, solids and stripes.
